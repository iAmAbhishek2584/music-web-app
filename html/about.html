About the Web Page

Project

  • This is a front end design of a Music Player Website. This websites displays how we can manipulate audios files inside a web page.
  • In this website we can play a song , we can go between previous and next song and scroll between a song.
  • This is website is fully responsive and can adapt to different screen sizes like mobile, tablet and desktops

Technologies Used

  • This front end design is build using HTML5 , CSS and Java script
  • The main functioning that is interacting with the audio was achieved with the use of java script Web API
  • This web page almost made from scratch without using any framework like bootsrap or tailwind CSS
  • The icons used in the webpage are imported from font awesome website using their cdnjs link in our webpage
